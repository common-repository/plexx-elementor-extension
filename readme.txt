Provides a quick and easy method of adding Plexx theme elements to a Page with Elementor plugin.

== Description ==

The Plexx Elementor Extension plugin contains 10 widgets for Plexx theme for usage with Elementor Free page builder plugin in less time with no coding required. The plugin works only with Plexx template which you can find on ThemeForest.

== Screenshots ==

1. The Plexx Elementor Extension widgets will appear in your list of Elementor widgets.
2. Drag any of it to a Page area.
3. Make changes and click Update.

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Plexx Elementor Extension" into the Search input and click the "Search Widgets" button.
3. Locate the Plexx Elementor Extension in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.
5. Navigate to Appearance &rarr; Widgets and [create a new instance](http://codex.wordpress.org/WordPress_Widgets#Activate_Widgets).

It can also be installed manually.

1. [Download](https://wordpress.org/extend/plugins/plexx-elementor-extension/) the plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Plexx Elementor Extension" in the list and click the "Activate" link.
5. Navigate to Appearance &rarr; Widgets and [create a new instance](http://codex.wordpress.org/WordPress_Widgets#Activate_Widgets).
